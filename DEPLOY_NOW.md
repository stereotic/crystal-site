# 🚀 ДЕПЛОЙ ЗА 5 МИНУТ - ПРОСТО СЛЕДУЙ ИНСТРУКЦИЯМ

## ЧТО НУЖНО СДЕЛАТЬ:

### 1. Запусти скрипт деплоя

Открой терминал (PowerShell или CMD) в папке проекта и выполни:

```bash
deploy-railway.bat
```

### 2. Следуй инструкциям на экране

Скрипт сам:
- ✅ Установит Railway CLI
- ✅ Авторизует тебя (откроется браузер)
- ✅ Создаст проект
- ✅ Добавит все переменные окружения
- ✅ Задеплоит проект

### 3. После деплоя

Скрипт выдаст URL типа: `https://crystal-site-production.up.railway.app`

Добавь переменную DOMAIN:
```bash
railway variables set DOMAIN=https://твой-url.up.railway.app
```

Настрой Telegram webhook:
```bash
curl -X POST "https://api.telegram.org/bot8184664856:AAGzpVyvWIKuBuc8OFZE1dl1GAzd__ohBbM/setWebhook" -d "url=https://твой-url.up.railway.app/webhook/telegram-bot"
```

### 4. Готово!

Открой URL в браузере - сайт работает!

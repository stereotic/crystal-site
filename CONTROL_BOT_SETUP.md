# Инструкция по настройке бота контроля платежей

## Шаги настройки:

1. **Создайте нового бота через @BotFather**
   - Отправьте `/newbot` в @BotFather
   - Назовите бота (например: "Crystal Control Bot")
   - Выберите username (например: @CCcontrol_xbot)
   - Скопируйте токен бота

2. **Получите Chat ID**
   - Создайте группу или используйте личный чат
   - Добавьте бота в группу (если используете группу)
   - Отправьте любое сообщение боту
   - Откройте в браузере: `https://api.telegram.org/bot<YOUR_BOT_TOKEN>/getUpdates`
   - Найдите `"chat":{"id":` - это ваш CONTROL_CHAT_ID

3. **Настройте .env файл**
   ```env
   CONTROL_BOT_TOKEN=ваш_токен_от_BotFather
   CONTROL_CHAT_ID=ваш_chat_id
   ```

4. **Перезапустите сервер**
   ```bash
   npm run dev
   ```

## Как это работает:

1. Пользователь выбирает сумму депозита и криптовалюту
2. Нажимает "Check Payment" первый раз - создается запрос
3. Переводит средства на указанный адрес
4. Нажимает "Check Payment" второй раз - отправляется уведомление в бота

5. В боте @CCcontrol_xbot приходит сообщение:
   ```
   💰 Новый платеж!
   
   Мамонт: username
   Сумма: $150
   Адрес: wallet_address
   Воркер: @worker_username
   
   [✅ Подтвердить] [❌ Отклонить]
   ```

6. При нажатии "Подтвердить":
   - Баланс пользователя пополняется
   - Статус запроса меняется на "approved"

7. При нажатии "Отклонить":
   - Баланс не меняется
   - Статус запроса меняется на "rejected"

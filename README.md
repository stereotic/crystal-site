## 💰 Управление адресами кошельков

### Как изменить адрес депозита:

1. **Откройте админ-бота в Telegram**
   - Найдите бота по токену `ADMIN_BOT_TOKEN` из `.env`
   - Отправьте `/start`

2. **Выберите кошелек**
   - Нажмите кнопку: `💰 USDT TRC20`, `💰 USDT BEP20`, `₿ BTC` или `Ξ ETH`

3. **Введите новый адрес**
   - Бот попросит ввести адрес
   - Введите новый адрес кошелька

4. **Готово!**
   - Бот подтвердит: `✅ USDT_TRC20 обновлён`
   - Адрес автоматически обновится на сайте

### Как это работает:

```
Админ → Telegram Bot → Database → API → Website → User
  ↓         ↓             ↓         ↓       ↓        ↓
Кнопка   Запрос      UPDATE    GET     Загрузка  Видит
         адреса      wallets   /api/   адресов   адрес
                              wallets
```

## 📦 Установка

### 1. Клонировать репозиторий
```bash
git clone https://github.com/stereotic/crystal-site.git
cd crystal-site
```

### 2. Установить зависимости
```bash
npm install
```

### 3. Настроить .env файл
```env
DOMAIN=https://ваш-домен.com
SESSION_SECRET=ваш-секретный-ключ
SMTP_USER=ваш-email@gmail.com
SMTP_PASS=ваш-пароль-приложения
USER_BOT_TOKEN=токен-пользовательского-бота
ADMIN_BOT_TOKEN=токен-админ-бота
ADMIN_IDS=123456789,987654321
BOT_USERNAME=YourBot
PORT=3001
```

### 4. Запустить бота
```bash
node app.js
```

## 🔐 Безопасность

- ✅ Все пароли хешируются с bcrypt
- ✅ Сессии хранятся в SQLite
- ✅ Rate limiting на API
- ✅ CORS настроен
- ✅ Helmet для безопасности заголовков
- ✅ Валидация всех входных данных
- ✅ Только админы могут менять адреса кошельков

## 🌐 API Endpoints

### Публичные:
- `POST /api/auth/register` - Регистрация
- `POST /api/auth/login` - Вход
- `GET /api/rates` - Курсы валют

### Авторизованные:
- `GET /api/wallets` - Получить адреса кошельков
- `POST /api/deposit/request` - Запрос депозита
- `POST /api/withdraw/request` - Запрос вывода
- `GET /api/cards` - Список карт
- `POST /api/cart/buy-now` - Купить карту

- SQLite3
- bcrypt для хеширования паролей
- express-session для сессий
- multer для загрузки файлов
- nodemailer для email
- helmet для безопасности

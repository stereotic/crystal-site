# Crystal Cards - Elite Marketplace

Полноценный веб-сайт для продажи карт с бэкендом на Node.js.

## Установка

1. Установите зависимости:
```bash
npm install
```

## Запуск

```bash
npm start
```

Сервер запустится на http://localhost:3000

## Функционал

- ✅ Регистрация и авторизация пользователей
- ✅ Покупка карт
- ✅ Пополнение и вывод средств
- ✅ Premium подписка ($220)
- ✅ Premium чат
- ✅ Проверка баланса карт (только для Premium)
- ✅ Поддержка с загрузкой файлов
- ✅ База данных SQLite

## Технологии

- Node.js + Express
- SQLite (better-sqlite3)
- bcrypt для хеширования паролей
- express-session для сессий
- multer для загрузки файлов

## Демо данные

При первом запуске автоматически создаются:
- 6 демо карт разных типов
- Кошельки для криптовалют (BTC, ETH, USDT)

## Структура

- `server.js` - основной сервер
- `index.html` - фронтенд
- `database.db` - база данных SQLite
- `uploads/` - загруженные файлы

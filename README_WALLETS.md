# 💰 Система управления адресами кошельков

## ✅ Что сделано:

Реализован полный функционал для изменения адресов депозита через админ-бота Telegram с автоматическим обновлением на сайте.

### Компоненты:

1. **База данных** - таблица `wallets` с 4 валютами:
   - BTC
   - ETH
   - USDT TRC20
   - USDT BEP20

2. **Админ-бот** - клавиатура с кнопками:
   - 💰 USDT TRC20
   - 💰 USDT BEP20
   - ₿ BTC
   - Ξ ETH

3. **API** - эндпоинт `/api/wallets` возвращает все адреса

4. **Сайт** - автоматически загружает и отображает адреса

---

## 🚀 Как использовать:

### 1. Откройте админ-бота в Telegram
- Найдите бота по токену `ADMIN_BOT_TOKEN` из `.env`
- Отправьте команду `/start`
- Увидите клавиатуру с 4 кнопками

### 2. Измените адрес кошелька
1. Нажмите на нужную кнопку (например, **💰 USDT TRC20**)
2. Бот попросит: `Введите новый адрес для USDT_TRC20:`
3. Введите адрес (например: `TN3W4H6rK2ce4vX9YnFQHwKENnHjoxb3m9`)
4. Бот подтвердит: `✅ USDT_TRC20 обновлён`

### 3. Проверьте на сайте
1. Откройте сайт в браузере
2. Войдите в аккаунт
3. Перейдите: **Profile** → **Deposit**
4. Выберите метод оплаты (BTC, ETH, USDT TRC20, USDT BEP20)
5. **Адрес обновится автоматически!**

---

## 🔄 Как это работает:

```
Админ нажимает кнопку в боте
         ↓
Бот запрашивает новый адрес
         ↓
Админ вводит адрес
         ↓
База данных обновляется (UPDATE wallets)
         ↓
Сайт загружает адреса (GET /api/wallets)
         ↓
Пользователь видит новый адрес
```

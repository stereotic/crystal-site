## 🚀 ИНСТРУКЦИЯ ПО ЗАПУСКУ:

### 1. Запустить бот:
```bash
cd "C:\Users\Артем\Desktop\projects\BLACK-BET TEAM\Crystalix app bot"
node app.js
```

### 2. Открыть админ-бота в Telegram:
- Найти бота по токену из `.env` (ADMIN_BOT_TOKEN)
- Отправить `/start`
- Увидите клавиатуру:
  ```
  [💰 USDT TRC20] [💰 USDT BEP20]
  [₿ BTC]         [Ξ ETH]
  ```

### 3. Изменить адрес:
1. Нажать на кнопку (например, **💰 USDT TRC20**)
2. Бот ответит: `Введите новый адрес для USDT_TRC20:`
3. Ввести адрес: `TYourNewAddressHere123456789`
4. Бот подтвердит: `✅ USDT_TRC20 обновлён`

### 4. Проверить на сайте:
1. Открыть: `http://ваш-домен.com` (или локально)
2. Войти в аккаунт
3. Перейти в **Profile** → нажать **Deposit**
4. Выбрать метод (например, USDT TRC20)
5. **Адрес обновится автоматически!**

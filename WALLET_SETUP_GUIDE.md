# 🎯 Руководство по настройке адресов кошельков

## ✅ Что уже сделано:

1. **База данных инициализирована** - таблица `wallets` создана с 4 валютами:
   - BTC
   - ETH  
   - USDT_TRC20
   - USDT_BEP20

2. **Админ-бот настроен** - есть 4 кнопки для изменения адресов:
   - 💰 USDT TRC20
   - 💰 USDT BEP20
   - ₿ BTC
   - Ξ ETH

3. **API работает** - эндпоинт `/api/wallets` возвращает все адреса

4. **Сайт подключен** - автоматически загружает адреса при открытии модального окна депозита

## 🚀 Как использовать:

### Шаг 1: Запустить бот
```bash
node app.js
```

### Шаг 2: Открыть админ-бота в Telegram
- Найдите вашего админ-бота
- Отправьте команду `/start`
- Вы увидите клавиатуру с 4 кнопками

### Шаг 3: Изменить адрес кошелька
1. Нажмите на нужную кнопку (например, **💰 USDT TRC20**)
2. Бот попросит ввести новый адрес
3. Введите адрес кошелька (например: `TN3W4H6rK2ce4vX9YnFQHwKENnHjoxb3m9`)
4. Бот подтвердит: `✅ USDT_TRC20 обновлён`

### Шаг 4: Проверить на сайте
1. Откройте сайт: http://ваш-домен.com
2. Войдите в аккаунт
3. Перейдите в профиль
4. Нажмите кнопку **Deposit**
5. Выберите метод оплаты (BTC, ETH, USDT TRC20, USDT BEP20)
6. **Адрес автоматически обновится!**

## 🔄 Как это работает:

1. **Админ нажимает кнопку** → Бот запрашивает новый адрес
2. **Админ вводит адрес** → Бот обновляет запись в таблице `wallets`
3. **Пользователь открывает депозит** → Сайт делает запрос к `/api/wallets`
4. **API возвращает адреса** → Сайт отображает актуальный адрес

## ⚠️ Важно:

- Адреса обновляются **мгновенно** - не нужно перезапускать сервер
- Каждая валюта хранится отдельно в базе данных
- Сайт загружает адреса при каждом открытии модального окна
- Только админы (указанные в `ADMIN_IDS` в `.env`) могут менять адреса
